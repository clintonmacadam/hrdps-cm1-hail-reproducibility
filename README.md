This repository contains input soundings and namelist.input files used for CM1 simulations, as well as all plotting scripts used for the manuscript "Hail Forecasting with an Operational 
Convective-Scale Model:  Assessment of Current Status and Examination of Potential Improvements". The provided namelist.input files are configured to run the two-moment configuration of the latest 
(v5) P3 at a 1250m horizontal grid spacing, with the umove/vmove parameters set for each case to keep the simulated storms roughly centered in the domain. Please contact Clinton Macadam at 
macadamc@myumanitoba.ca if you have any questions regarding the provided files and/or the associated manuscript.
